# Features
- **Authentication**: Secure login/signup.
- **Interactive Courses**: Python, C, Java courses with progress tracking.
- **Gamification**: XP system, levels, badges, and leaderboard.
- **AI Chatbot**: 24/7 coding assistance powered by Gemini.
- **AI Quizzes & DSA**: Dynamically generated quizzes and LeetCode recommendations.
- **Mini-Games**: Code Dino Extreme.
