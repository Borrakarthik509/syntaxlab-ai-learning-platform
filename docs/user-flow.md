# User Flow
1. **Landing Page**: User visits `index.html`.
2. **Authentication**: User logs in or signs up via `loginsignup.html`.
3. **Dashboard**: User is redirected to `afterloginhome.html` (Home) or `userdashboard.html` (Profile).
4. **Learning**: User browses `courses.html` and tracks XP.
5. **AI Zone**: User explores `AIzonehome.html`, uses Chatbot, gets DSA recommendations, or takes quizzes.
